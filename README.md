# .github Repository for MunyayLLC

This is a special `.github` repository that provides default community health files and configurations for all repositories in the MunyayLLC organization.

## What's Included

- **FUNDING.yml**: Sponsorship and funding configuration
- **SUPPORT.md**: Support and contact information
- **Issue Templates**: Standardized issue templates for the organization

These files will be automatically used as defaults across all repositories in the organization that don't have their own versions.
